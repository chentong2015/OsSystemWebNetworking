1. clear
   clears the contents of your terminal display

2. rm
   delete file

3. chown 修改文件或者目录的所有权ownership
   change the ownership of a file

4. pwd
   Shows you what directory you are in

5. mkdir
   creates an empty directory

6. whoami
   displays your current username

7. history
   display your previous commands

8. grep
   search for strings inside a file

9. usermod 修改指定的用户名称(user modify)
   "usermod -g canoc cabasa" is used to modify the "cabasa" user and change its group to "canoc"
