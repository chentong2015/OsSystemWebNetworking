Linux服务器安装java环境(ubuntu)
从官网下载java的包，并解压到定义的目录中
> tar zxvp xx.tar
> sudo gedit /etc/profile 添加环境变量到profile文件
  export JAVA_HOME="path that you found"
  export PATH=$JAVA_HOME/bin:$PATH
> source /etc/profile 再编译, 立即执行更新
> java -version

直接使用默认的安转即可
>sudo apt-get update
>sudo apt-get install oracle-java7-installer
>sudo apt install openjdk-8-jdk

查看和修改tomcat分配给java JVM运行的空间大小
2.1 $ java -XX:+PrintFlagsFinal -version | grep -iE 'HeapSize|PermSize|ThreadStackSize'
	uintx InitialHeapSize                          := 64781184        {product}
	uintx MaxHeapSize                              := 1038090240      {product}
	uintx PermSize                                  = 21757952        {pd product}
	uintx MaxPermSize                               = 174063616       {pd product}
	intx ThreadStackSize                            = 1024            {pd product}
	java version "1.7.0_51"
	OpenJDK Runtime Environment (IcedTea 2.4.4) (7u51-2.4.4-0ubuntu0.13.10.1)
	OpenJDK 64-Bit Server VM (build 24.45-b08, mixed mode)
2.2 创建/tomcat/bin/setenv.sh文件
	export CATALINA_OPTS="-Xms512M -Xmx1024M"
2.3 直接编辑catalina.sh
	JAVA_OPTS="-server -Xms128m -Xmx512m -XX:PermSize=256m -XX:MaxPermSize=512m"
2.4 编辑/etc/init.d/tomcat
	HEAP="-Xms2048m -Xmx2048m"
	export JAVA_OPTS="$HEAP"
