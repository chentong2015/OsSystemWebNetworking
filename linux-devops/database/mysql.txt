安装对应的Mysql数据库到系统的默认位置
mysql> sudo apt update
mysql> sudo apt-get install mysql-server
mysql> sudo mysql_secure_installation 设置数据库的安全属性保护
mysql> sudo mysql
mysql> SELECT * FROM mysql.user
